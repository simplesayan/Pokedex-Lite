# Pokedex Lite
A responsive web application that allows users to explore Pokémon data using the PokéAPI.  
Users can search, filter, paginate, view details, and mark their favorite Pokémon.

# Core Features
# Search Pokémon
  - Real-time search by name
# Filter by Type
  - Filter Pokémon based on types like Fire, Water, Grass, etc.
# Pagination
  - Navigate through Pokémon using Next/Previous controls
# Favorites
  - Mark/unmark Pokémon as favorites
  - Persisted using localStorage
# Detailed View
  - View stats like HP, Attack, Defense, and types in a modal
# Responsive UI
  - Works across mobile, tablet, and desktop screens
# Loading & Error Handling
  - Graceful UI feedback during API calls

# Tech Stack
# Frontend: React (Vite)
# HTTP Client: Axios
# State Management: React Hooks (useState, useEffect)
# Styling: CSS (custom responsive design)
# API: PokéAPI (https://pokeapi.co/)

# Installation & Setup
Clone the Repository ||
git clone <your-repo-link> ||
cd pokedex-lite ||
npm install ||
npm run dev
